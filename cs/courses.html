A list of textbooks and course material for Computer Science courses. This webpage is an evolving document for my personal reference.

My CSE bookshelf

  • The C Programming Language. Kernighan, Ritchie. Second edition
  • Computer Architecture: A Quantitative Approach. Hennessey, Patterson. Fifth edition. This book is so comprehensive!
  • Operating Systems: Three Easy Pieces. Arpaci-Dusseau.
  • Operating System Concepts. Silberschatz, Galvin, Gagne. 9th edition.
  • Introduction to Algorithms. Cormen, Leiserson, Rivest, Stein. Third edition.
  • Introduction to the Theory of Computation. Sipser. Third edition.
  • Elements of Information Theory. Thomas, Cover. Second edition.
